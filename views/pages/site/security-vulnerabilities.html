Security Vulnerabilities

To receive all aah framework security announcements is to subscribe to the aah security list. The mailing list is very low traffic, and receives notifications only after Security reports have been managed by the dev team and fixes are publicly available.

Reporting Vulnerabilities

I strongly encourage people to report such problems to our private security email address first, before disclosing them in a public forum.

All security bugs in aah framework should be reported by email to security@aahframework.org.

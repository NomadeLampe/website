aah framework

A scalable, performant, rapid development Web framework for Go

Get started with aah

Requires ≥ go1.8

aah framework makes it easy to build scalable web apps and mobile apps API with Go language. aah is scalable, performant and rapid development. aah values DRY and convention over configuration. It makes application development more fun, free and open source.

aah framework goal is to achieve rapid, highly configurable and balanced Go web framework, from micro service to small, medium and enterprise scale.

Configuration

aah configuration is similar to Typesafe HOCON syntax. Flexible and easy to configure. aah is opinionated and defaults to set of conventions and configurations know more.

Scalable

aah scales well with your use case. You can build scalable web apps and mobile apps APIs comfortably. Framework is flexible, well structured and ability to fulfil modern application needs.

High Performance

Optimized routing on domain and sub-domains, less GC overhead, optimized middlewares and pragmatic design.

Reply

aah supports various HTTP reply like HTML (layouts), JSON, JSONP, XML, Text, File (attachment, inline) and Bytes. Static file delivery via `http.ServeContent` and directory listing know more.

Domain & Subdomains

Developer Friendly

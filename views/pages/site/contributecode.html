Contribute to Code

aah framework's codebase, modules and documentation is hosted on Github. It's easy to get started with aah codebase. If you have any questions, you can ask on the mailing list or you can also chat on gitter.im with aah community.

To keep aah codebase arranged neatly and to make sure we work together smoothly there are some Contributor Guidelines that you'll need to make sure you follow.

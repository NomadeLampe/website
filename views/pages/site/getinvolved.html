Get Involved

aah framework is an open source project and you can become a part of the ongoing development. There are plenty of ways you can contribute.

Join the Mailing List

The aah framework mailing list is where the aah community meets. Asking and answering questions on the mailing list is a great way to share knowledge about aah framework.

Join the Gitter Chat

Gitter is a real time chat channel like IRC. The aahframework/community channel is used by aah users to discuss the ins and outs of writing great aah applications.

Contribute to Stack Overflow

Stack Overflow is a Q&A site for programmers. Stack Overflow has a very active community of people asking and answering questions about aah. Users can vote on each others contributions and earn reputation points.

Report Bugs

Bug report helps aah framework improve, so please report any issues you have! aah uses GitHub to track issues. When you report bugs, make sure you include lots of detail, including reproducible tests, example code or anything else you think might help.

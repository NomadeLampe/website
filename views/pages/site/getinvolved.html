Get Involved

aah framework is an open source project and you can become a part of the ongoing development. There are plenty of ways you can contribute.

Please Note:

  • For general usage questions and need clarification, posting a question on StackOverflow is the way to go
  • Want to discuss with aah development team. Join and discuss on Mailing List and Gitter Chat is the way to go

   Contribute to Stack Overflow

Stack Overflow is a Q&A site for programmers. Stack Overflow has a very active community of people asking and answering questions about aah. Users can vote on each others contributions and earn reputation points.

   Documentation & Cookbook

You're encouraged to improve, added documentation and provide cookbook to aah framework. Even if it's small it really helps your fellow developers. For example: you can correct typo mistakes, grammer, provide documentation translation, etc.

   Join the Mailing List

The aah framework mailing list is a Google Groups, it's a great place to discuss with aah dev team. Join the mailing list to start your conversation. The mailing list is for discussion around the framework's development. For questions we recommend StackOverflow.

   Join the Gitter Chat

Gitter is a real time chat channel like IRC.

  • The aahframework/community channel is used by aah community.
  • The aahframework/dev-team channel is used by aah dev team.

   Report Bugs

The aah framework uses GitHub to track issues. When you report bugs, make sure you include lots of detail, including reproducible tests, example code or anything else you think might help. So please report any issues you have!

   Report Security Vulnerabilities

I strongly encourage people to report such problems to our private security email address, before disclosing them in a public forum. Please click here to read more information.

   Contribute to Code

Would you like to contribute to codebase and you know the area you want to contribute to, this is great; just click here to know more.
